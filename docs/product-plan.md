# Product Plan — Startup Radar MVP

## 1. Problème utilisateur

Les investisseurs non institutionnels, business angels, family offices, incubateurs et entrepreneurs ont accès à beaucoup d'information sur les startups, mais cette information est fragmentée :

- bases de données startup ;
- articles analystes ;
- GitHub ;
- LinkedIn ;
- Product Hunt ;
- arXiv ;
- communiqués de levée ;
- pitch decks ;
- rapports VC ;
- signaux terrain.

Le problème est de transformer ces signaux en décision structurée : `Pass`, `Watch`, `Deep Dive`, `Investable`.

---

## 2. Utilisateurs cibles

### Utilisateur interne

Investisseur-opérateur souhaitant analyser les startups proches de ses domaines d'expertise.

### Utilisateurs futurs

- business angels ;
- family offices ;
- incubateurs ;
- fonds VC early stage ;
- corporate innovation ;
- banques privées ;
- agences publiques d'innovation ;
- cabinets de conseil.

---

## 3. Fonctionnalités MVP

### 3.1 Startup Database

Créer, importer et enrichir des fiches startups.

Champs minimaux :

- nom ;
- pays ;
- secteur ;
- sous-secteur ;
- site web ;
- LinkedIn ;
- stade ;
- montant levé ;
- investisseurs ;
- fondateurs ;
- produit ;
- clients ;
- signaux faibles ;
- score global.

### 3.2 Startup Screener

Filtres :

- pays : France, Suisse, Algérie, Europe, MENA, global ;
- secteur ;
- stade ;
- financement ;
- maturité produit ;
- score ;
- décision.

### 3.3 Scoring Engine

Scores principaux :

- Market Score ;
- Product Score ;
- Technical Moat Score ;
- Traction Score ;
- Team Score ;
- Valuation Score ;
- Risk Score ;
- Kamel Edge Score.

### 3.4 Due Diligence Memo

Génération d'une fiche d'une page :

1. résumé ;
2. problème ;
3. solution ;
4. marché ;
5. technologie ;
6. traction ;
7. business model ;
8. valorisation ;
9. risques ;
10. décision.

### 3.5 Watchlist

Classement en listes :

- Top 10 à suivre ;
- Top 5 à analyser ;
- Top 5 investissables ;
- Top 5 survalorisées ;
- Top 5 fausses IA potentielles.

---

## 4. MVP non inclus

Le MVP ne doit pas encore faire :

- conseil financier personnalisé ;
- gestion de portefeuille client ;
- exécution d'ordres ;
- recommandation d'achat automatisée ;
- scraping agressif non conforme aux CGU ;
- prédiction absolue de valorisation.

---

## 5. Produit vendable futur

### Offre 1 — Rapport mensuel

`Radar IA & Deeptech — Top tendances, startups et signaux de valorisation.`

### Offre 2 — Due diligence technique

Analyse technique d'une startup pour business angel ou family office.

### Offre 3 — SaaS Radar

Plateforme avec base startup, scores, alertes et fiches automatisées.

---

## 6. Indicateurs de succès MVP

- 100 startups qualifiées ;
- 20 fiches deep dive ;
- 10 sources de données structurées ;
- 1 score startup opérationnel ;
- 1 dashboard ;
- 1 rapport Top 10 généré ;
- 3 décisions d'investissement mieux documentées.
