# Investment Thesis — Physical AI / Robotics

## 1. Pourquoi ajouter ce segment

La prochaine vague de l'IA pourrait passer de l'IA logicielle vers l'IA incarnée : robots humanoïdes, manipulation robotique, robot foundation models, entrepôts autonomes, usines et services.

Ce segment doit être suivi dans le radar parce qu'il combine :

- modèles IA multimodaux ;
- vision-language-action models ;
- robot foundation models ;
- capteurs et actionneurs ;
- données de téléopération ;
- manufacturing hardware ;
- marchés industriels avec forte pénurie de main-d'oeuvre.

## 4. Critères de scoring spécifiques

### Signaux positifs

- clients industriels réels ;
- pilotes payants ;
- coût robot en baisse ;
- capacité de production ;
- données propriétaires issues de téléopération ;
- partenariats avec industriels ou hyperscalers ;
- modèle robotique réutilisable sur plusieurs embodiments ;
- amélioration rapide du taux d'autonomie.

### Red flags

- autonomie réelle faible ;
- dépendance excessive à la téléopération ;
- hardware trop coûteux ;
- gross margin incertaine ;
- cycles de certification longs ;
- sécurité en environnement humain non prouvée ;
- valorisation très élevée sans revenus proportionnels ;
- besoin massif de capital pour manufacturing.

---

## 5. Questions de due diligence

1. Quel est le taux réel d'autonomie hors démonstration ?
2. Quelle part du travail est encore téléopérée ?
3. Quel est le coût complet par robot ?
4. Quel est le coût de maintenance ?
5. Quelle est la marge brute visée ?
6. Existe-t-il des clients payants ?
7. Le robot résout-il un problème industriel urgent ?
8. La startup possède-t-elle des données propriétaires ?
9. Le logiciel peut-il être déployé sur plusieurs robots ?
10. La valorisation laisse-t-elle encore un potentiel x5 / x10 ?

---

## 6. Positionnement dans le radar

Le segment doit être suivi comme thèse prioritaire, mais avec un risque élevé.

Recommandation de traitement :

- classer séparément les startups Physical AI ;
- ne pas mélanger avec AI software pur ;
- appliquer une pénalité risque plus forte ;
- exiger des preuves de traction industrielle ;
- distinguer démonstration marketing et autonomie réelle.

---

## 7. Décision d'investissement indicative

Physical AI peut produire des valorisations très élevées, mais le risque capitalistique est supérieur à l'IA logicielle.

Approche recommandée :

- surveiller beaucoup ;
- investir petit ;
- privilégier software robotics et robot foundation models ;
- être prudent sur les fabricants hardware très capitalistiques ;
- chercher les acteurs avec données propriétaires et clients industriels.
